# VS Code Marketplace publishing

The VS Code extension is packaged as `diagram-as-code-vscode-<version>.vsix` by the product release. Its Marketplace publisher is `hnamkk`; the publisher must be created and verified in the Visual Studio Marketplace before the first publication.

1. Create or verify the `hnamkk` publisher and restrict its ownership to the release maintainers.
2. Create a Visual Studio Marketplace personal access token with only the `Marketplace (Manage)` scope.
3. Store it as `VSCE_PAT` in the protected GitHub `product-release` environment.
4. Install the release VSIX on a clean VS Code profile before publishing.
5. Publish only a versioned VSIX generated by the tagged Product Release workflow.

The release workflow skips Marketplace publication when `VSCE_PAT` is absent. GitHub Release artifacts remain available for self-hosted installation in that case.
